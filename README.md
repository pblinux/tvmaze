# tvmaze
A simple app (and challenge) to consume TV Maze API

## Geting started
Builted with Android Studio Artic Fox | 2020.3.1 Beta 3

### Stack
  * Kotlin 1.5.10
  * Jetpack Compose
  * Hilt
  * Paging
  * Navigation
  * Room

### Comments
  * Sadly, Navigation for Compose not support transitions yet :( Hope to change this in the future.
